# 🍽 Recipe Website

## 📌 Description
A fully functional recipe website built using HTML, CSS, and JavaScript. 
It fetches real recipes from TheMealDB API and allows users to search, view details, and save favourites.

## 🚀 Features
- Homepage with recipe cards
- Search functionality
- Recipe details page
- Favourite recipes (LocalStorage)
- Dark mode
- Fully responsive design

## 🔧 Technologies Used
- HTML
- CSS
- JavaScript
- TheMealDB API

## ▶️ How to Run
1. Download the project files
2. Open index.html in your browser
